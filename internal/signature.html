Setup Instructions

For Gmail:

  1. Select the signature table above (click and drag)
  2. Copy (Cmd + C on Mac, Ctrl + C on Windows)
  3. Go to Gmail → Settings (gear icon) → See all settings
  4. Scroll to "Signature" section
  5. Create new or edit existing signature
  6. Paste (Cmd/Ctrl + V)
  7. Replace [Your Name], [Your Title], and [your email] with your info
  8. Save Changes

For Apple Mail:

  1. Select and copy the signature
  2. Mail → Settings → Signatures
  3. Click + to add new signature
  4. Paste and edit your details

For Outlook:

  1. Select and copy the signature
  2. File → Options → Mail → Signatures
  3. Click New, name it, paste into editor
  4. Edit your details and save
