## What changed

The Servantium marketing website and help documentation have been rebuilt from the ground up using **Verdant** — our custom design system built on Astro 6 and Tailwind CSS v4.

### Marketing site
- Migrated from static HTML to Astro with Tailwind CSS
- Self-hosted fonts (no more Google Fonts external requests)
- Foxi Pro-inspired page animations and scroll reveals
- 30 blog posts converted to MDX with full-featured listing (search, filters, pagination)
- Author profiles, category pages, RSS feed

### Help documentation
- Replaced Starlight with **Grove** — our custom documentation framework
- Tailwind v2 docs-inspired layout (sidebar, TOC, clean typography)
- 9 comprehensive feature guides covering every Servantium capability
- Release notes timeline with automated pipeline
- Full-text search via Pagefind

### Infrastructure
- Tailwind CSS v4 via Vite plugin
- React integration for Cal.com booking component
- GitHub Actions CI build check
- Automated release notes generation via Claude API
- Security headers with Content Security Policy
- Auto-generated sitemap

## Why this matters

- **Faster page loads** — self-hosted fonts, build-time optimization, Cloudflare CDN
- **Better content pipeline** — drop an MDX file in GitHub, it's live in 45 seconds
- **Unified design** — same Nav, Footer, and tokens across marketing and docs
- **Searchable everything** — Pagefind indexes all 62 pages
