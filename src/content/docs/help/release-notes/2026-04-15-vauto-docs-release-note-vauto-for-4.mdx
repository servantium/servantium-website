## What changed

The What's New page now displays help cards with white backgrounds, rounded borders, floating version badges, and hover lift effects. Three new astronaut variants launched: accountant (briefcase and ledger), controller (hi-vis vest and signal wand), and explorer (telescope and map). Each variant corresponds to a platform section: detective for Scope, accountant for Price, controller for Staff, professor for Learn, and explorer for About. The Resources mega-menu consolidated Calculators, Templates, and Checklists into a single Templates entry. Publishing pipeline documentation received a complete rewrite identifying named actors and workflow steps with updated diagrams.

## Why it matters

- Cleaner card design matches familiar web patterns and improves scan-ability
- Distinct astronaut identities help users recognize platform sections faster
- Fewer Resources menu items reduce navigation friction
- Pipeline docs now clearly show which system handles each step, simplifying troubleshooting

<Aside type="tip">Pipeline docs warn that direct pushes to develop bypass automated release note and doc update workflows.</Aside>
