## What changed

Help cards on the What's New page now use Tailwind's centered-icon pattern: white cards with rounded borders, floating version badges, and hover lift effects. Three new astronaut variants shipped: accountant (briefcase and ledger), controller (hi-vis vest and signal wand), and explorer (telescope and map). These match platform sections: detective for Scope, accountant for Price, controller for Staff, professor for Learn, and explorer for About. The Resources mega-menu collapsed Calculators, Templates, and Checklists into a single Templates entry. Publishing pipeline docs received a complete rewrite with named actors—GitHub Actions, Servantium Bot, ReleaseWriter, Help Docs Orchestrator, Help Docs Writer, Internal Portal—and updated SVG diagrams.

## Why it matters

- Cleaner visual hierarchy on marketing pages matches industry patterns visitors expect
- Astronaut variants give each platform section a distinct identity
- Simpler Resources navigation reduces clicks to downloadable tools
- Pipeline docs now explain who does what, making workflow failures easier to debug

<Aside type="tip">Pipeline docs include a "stealth release" warning: direct pushes to develop without a PR will not trigger automated release notes or doc updates.</Aside>
